Operating rule
Notion remains canonical. This interface should not become a second source-of-truth. It should display polished summaries, charts, and status pulled from Notion snapshots.
v0.9 is still a prototype. The next build should replace these hard-coded values with data from data/redbird-dashboard.sample.json, then eventually a real Notion export.
